Answers to questions:

  1. The box model says that a box is wrapped around every html element, this box contains margins, borders, padding, and then what's actually present.
  2. A block-level element is a html element that occupies the entire space of the parent container.
  3. a form is an example of a block level element
  4. An inline element is an elemt that doesn't start on a new line and takes as little width as possible.
  5. An example of an inline element is a button.
  6. You can include css internally or externally. Internally by using the style tag, externally by creating a new file called style.css
  7. A selector selects a specific HTML element that needs to be styled
  8. CSS rule to set the backround color black of the body element to: #0000FF is: body {background-color: #0000FF;}
